# v1.20 Promotion Decision

**Milestone:** v1.20 Runtime Sandbox Candidate and Exhibition Reliability Proof  
**Decision date:** 2026-05-25  
**Decision:** keep Python as non-counted exhibition beta; keep runtime isolation as readiness evidence with executable Docker/container candidate evidence.

## Decisions

- Python remains **non-counted exhibition beta**.
- Python is not eligible for ranked, ladder, counted, gauntlet, or broad production multi-language play.
- JS/TS support remains intact and remains the counted Strategy path.
- Docker/runc `container-subprocess` is the primary executable stronger sandbox candidate lane and passed local strict evidence, hostile probes, and signed-in proof participation.
- Docker/container evidence is **not production sandbox certification**.
- gVisor/runsc remains a fail-loud non-promotion record because `runsc` is unavailable locally and no runsc adapter executed probes.
- No arbitrary PyPI/package installs are promoted.
- No Strategy execution moved into web/API/Go.
- Go remains responsible for orchestration, persistence-facing behavior, Match lifecycle, scoring, retry/job lifecycle, and public evidence.
- Runtime service remains responsible for hostile Strategy execution only through schema-validated ABI envelopes and registered runtime implementations.
