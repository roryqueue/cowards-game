# v1.32 Rust Production Proof

## Promotion Decision

Rust is promoted to counted eligible Strategy support through the constrained `strategy-language-provider-rust-wasi` provider path.

## Evidence

- Runtime-service `/validate-strategy` owns Rust validation and build.
- Rust source is compiled to immutable `wasm32-wasip1` WASM/WASI Preview 1 artifacts.
- Artifact metadata records source hash, artifact hash/bytes, toolchain, target triple, WASI profile, ABI envelope/version, validation status, and public evidence.
- Runtime-service signs provider validation with an HMAC proof over provider id, contract version, source hash/bytes, and artifact hash/bytes.
- Counted-entry gates verify runtime metadata, provider proof, artifact metadata, decoded artifact bytes, target, WASI profile, ABI, and validation status.
- Runtime execution remains behind runtime-service / Runtime Broker / Wasmtime.
- Public result/evidence surfaces count provider-validated Rust as counted while preserving historical non-counted governance.

## Claims Not Made

- No Zig promotion is made by this artifact.
- No package ecosystem or external crate support is added.
- No direct-export, Component Model, or WIT ABI migration is made.
- No broad sandbox certification for arbitrary WASM/WASI programs is claimed.
