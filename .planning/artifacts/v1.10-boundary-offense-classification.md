## Classification Key

| Class | Meaning | v1.10 Handling |
| --- | --- | --- |
| Selected read slice | A read/user surface that v1.10 intends to migrate or disentangle. | Candidate for strict enforcement after service proof. |
| Deferred write/orchestration | A mutation, admin action, job/schedule action, or Match orchestration surface. | Remains report-only/out of scope. |
| Deferred private source/replay | Source-bearing or owner-debug/private Chronicle surface. | Remains report-only/out of scope. |
| Legacy facade | Broad web facade still owning mixed direct persistence behavior. | Shrinks only as selected callers move away; not hidden. |
| Test/type cleanup | Test or type import that should move only after production DTO ownership closes. | Candidate only when tied to a selected read slice. |
