# v1.16 Selected Go Route Manifest

This artifact records the Phase 105 selected normal web/API route families that must use Go-owned contracts without TypeScript service, persistence, or Chronicle fallback.

Machine-readable source: `.planning/artifacts/v1.16-selected-go-route-manifest.json`

## Selected Route Families

- Auth/session: session read, sign-in, sign-up, sign-out.
- Account Strategy Revisions: list, create/save, owner source.
- Account forks: Starter and Advanced forks.
- Exhibitions: selected MatchSet creation.
- Public reads: Strategy, player, ladder, MatchSet summary.
- Public replay: metadata and public evidence.
- Health: frontend health proxy/validation for strict Go mode.

## Fallback Policy

All selected entries use `no_typescript_backend_fallback` and stopped-Go behavior `fail_closed_classified`.

## Explicitly Out of Scope

Workshop, broader ladder mutations, governance/admin, owner-debug/private Chronicle migration, test-support routes, fixtures, rollback/parity paths, migrations/schema ownership, and runtime replacement remain non-normal for Phase 105.
