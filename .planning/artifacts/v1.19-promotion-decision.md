# v1.19 Promotion Decision

**Milestone:** v1.19 Runtime Isolation Readiness and Exhibition Beta Trust  
**Decision date:** 2026-05-25  
**Decision:** keep Python as non-counted exhibition beta; keep runtime isolation as readiness evidence only.

## Decisions

- Python remains **non-counted exhibition beta**.
- Python is not eligible for ranked, ladder, counted, or broad production multi-language play.
- JS/TS support remains intact and remains the counted Strategy path.
- Runtime isolation evidence is stronger, monitored, and more explicit, but it is **not production sandbox certification**.
- Docker/container and gVisor/runsc-style lanes remain readiness/availability evidence unless required live probe lanes genuinely pass.
- No arbitrary PyPI/package installs are promoted.
- No Strategy execution moved into web/API/Go.
- Go remains responsible for orchestration, persistence-facing behavior, Match lifecycle, scoring, and public evidence.
- Runtime service remains responsible for hostile Strategy execution only through schema-validated ABI envelopes and registered runtime implementations.
