## Model

`match_execution_quarantines` is private operator state. It references `match_jobs` and `matches`, stores a deterministic quarantine ID, records whether the terminal reason was `retry_exhausted` or `non_retryable_terminal`, and keeps redacted operator evidence.

The table does not introduce a public MatchSet/result/replay lifecycle state. Public projection remains behind `match-execution-app-v1`.

## Redaction

Operator evidence is allowlisted. Nested details reuse Go's existing `sanitizeMatchJobFailureDetails` helper. The model excludes Strategy source, StrategyMemory, SoldierMemory, objective payloads, raw diagnostics, stderr, stacks, host paths, env values, tokens, DB details, package paths, and private runtime internals.
