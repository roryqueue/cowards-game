# v1.14 Boundary Baseline

**Captured:** 2026-05-23  
**Milestone:** v1.14 Generic Strategy Artifact and Runtime Boundary Contract

## Baseline

v1.14 starts from the v1.13 selected Go backend cutover. Go owns selected public reads, auth/session routes, account revision list/source/create/save, and exhibition creation. TypeScript remains the parity oracle and rollback reference. The TypeScript worker/runtime-js boundary remains the only Strategy execution owner.

Boundary monitors start from:

- `strict_offenses=0`
- `report_only_offenses=29`

Any change to those counts must be explicit and justified.

## Ownership Categories

Machine-readable ownership facts are in `.planning/artifacts/v1.14-boundary-baseline.json`.

- Live Go routes: selected public reads, session/auth, account revisions, and exhibition creation.
- Fixture/parity routes: Go service fixtures and generated route manifests used for schema/privacy parity.
- TypeScript oracle/reference: Starter, Advanced, account revision, and service behavior used for parity comparison.
- Worker-owned runtime surfaces: `apps/worker` and `packages/runtime-js` execution paths.
- Deferred surfaces: job claiming/completion, Match execution, Chronicle generation, Go migrations, full replay projection, owner-debug migration, Workshop runtime ownership, production sandbox promotion, and counted non-JS play.
- v1.14 promotion candidates: generic Strategy artifacts, generated manifests, ABI v1.14, JS adapter conformance, Go Starter/Advanced forks, manifest-backed lineage, centralized privacy contract, and board realism gates.

## Non-Goals

v1.14 does not move hostile Strategy execution into Go, web, or API processes. It does not use Node `vm` as a security boundary. It does not promote counted non-JS play, production sandbox isolation, Go migrations, full replay projection, owner-debug replay migration, Workshop runtime ownership, or job claiming/completion migration.

Each non-goal is either monitored in Phases 93/95 or explicitly deferred to a future milestone in the JSON artifact.

## Privacy Baseline

Public/service/Go/topology/monitor outputs must omit Strategy source, StrategyMemory, SoldierMemory, objective payloads, owner debug, raw Awareness Grid, stack traces, stderr, sessions, tokens, host paths, DB DSNs, and private runtime internals by default.

Allowed source-returning exceptions are authenticated owner-private source retrieval and built-in forkable source classified explicitly by generated Strategy Artifact manifests.
