# v1.32 Public Evidence Language Pass

## Public Position

TypeScript, Python, Rust, and Zig are supported counted Strategy paths only through provider-compatible runtime evidence. Rust and Zig use immutable WASM/WASI Preview 1 stdin/stdout JSON artifacts. Provider proof is a counted-eligibility requirement, not broad sandbox certification.

## Privacy Position

Public results, replay evidence, and Learn/docs describe privacy exclusions without exposing private data. Public output omits Strategy source, StrategyMemory, SoldierMemory, objective payloads, raw diagnostics, host paths, environment values, credentials, and private runtime internals by default.

## No-Fallback Position

Runtime failures fail closed instead of falling back to another language or JS/TS path.

## Verification

Focused web/spec verification passed for Learn, evidence copy, result view model, replay client/state, and public fixture adapters.
