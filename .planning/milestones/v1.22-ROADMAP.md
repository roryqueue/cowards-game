## Execution Summary

v1.22 keeps the v1.21 Rust path as the regression floor while adding Zig only through a proof gate. Local `zig 0.16.0` is detected through PATH-aware preflight, Zig compiles a no-std WASI Preview 1 Strategy to immutable `wasm32-wasi` artifact metadata, and Wasmtime executes the same stdin/stdout JSON ABI used by Rust. WASM/WASI evidence now covers 19 probes, including Zig compile/runtime, resource limits, malformed output, invalid schema, stale hash, and forbidden capability gates.

## Decisions

- Zig is non-counted exhibition alpha only when compile, artifact, runtime, and ABI proof pass.
- WASM/WASI hardening remains candidate-readiness evidence, not production sandbox certification.
- v1.22 keeps WASI Preview 1 stdin/stdout JSON as the only execution ABI; direct exports and component model/WIT are deferred to a dedicated future proof.
- JS/TS remains the counted Strategy path. Python remains non-counted exhibition beta. Rust remains non-counted exhibition alpha.
