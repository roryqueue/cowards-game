# Phase 173 Plan

## Goal

Review, fix, decide claims, archive, commit, and tag v1.24.

## Scope

In scope:
- Code review for abuse-lab, ABI-spike, no-fallback, privacy, and boundary changes.
- Validation of requirement coverage, abuse evidence, readiness matrix, ABI decision, signed-in proof, replay plausibility, and public privacy.
- Final milestone decision on production sandbox certification, non-JS counted status, and active ABI.
- Archive planning artifacts, remove active requirements at milestone close, commit, and tag.

Out of scope:
- New feature work after closure except fixes required by audit findings.
- Retrospective promotion not supported by evidence.

## Tasks

1. Run code review and capture findings.
2. Run milestone validation against every v1.24 requirement.
3. Verify abuse-lab evidence, readiness matrix, ABI decision, signed-in proof, replay plausibility, privacy scans, and no-fallback gates.
4. Fix or explicitly document any closure-blocking gaps.
5. Write final promotion/sandbox/ABI decision artifacts.
6. Archive v1.24 planning artifacts and clear active requirements according to GSD milestone close conventions.
7. Commit and tag v1.24 with explicit evidence references.
