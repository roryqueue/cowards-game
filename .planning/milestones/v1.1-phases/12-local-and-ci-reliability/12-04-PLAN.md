# Plan 12-04: CI Command Split

## Objective

Add checked-in CI jobs that keep fast checks, service-backed E2E, and replay visual regression separate and locally reproducible.

## Tasks

1. Add root scripts for fast checks and explicit service/visual E2E slices.
2. Add `.github/workflows/ci.yml` with:
   - fast job: install, format, lint, typecheck, unit tests
   - service-e2e job: Postgres/Redis services, preflight, service-backed Workshop-to-replay E2E
   - visual-e2e job: deterministic replay fixture visual regression
3. Keep Playwright config compatible with both fixture-only and service-backed flows.

## Verification

- `pnpm test:fast`
- `pnpm e2e:smoke`
- `pnpm e2e:visual`
- `pnpm e2e:service`

## Done

CI and local scripts expose the same trust-loop slices without hiding service failures behind a monolithic command.
