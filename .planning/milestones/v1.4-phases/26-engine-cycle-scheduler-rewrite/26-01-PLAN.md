# Phase 26 Plan 01: Rewrite Engine Scheduler for Cycle Interleaving

## Objective

Move the pure rules engine from full-Activation sequencing to selected-slot
Cycle-layer scheduling under `cowards-rules-v1.4`.

## Tasks

1. Add explicit Activation slot state and terminal reasons.
2. Extract a reusable single-Cycle resolver that handles Cycle-start Backstab,
   SoldierBrain input, action resolution, Cycle-end Backstab, slot close, and
   skip events.
3. Rework `resolveRound` so selected slots are opened once and then stepped by
   Cycle layer in snake order.
4. Change blocked MOVE/PUSH so it consumes a Cycle but does not terminate the
   selected slot or count as Advance.
5. Add focused tests for Round 3 interleaving, blocked movement continuation,
   no-Advance cleanup, runtime failure, and match-end interruption.
