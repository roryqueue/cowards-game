## Audit Findings Fixed

| Finding | Resolution |
| --- | --- |
| ABI bridge signed/transmitted transpiled executable source instead of preserving the original revision source/hash in the ABI envelope. | ABI request envelopes now use original revision source text and hash for source validation while passing transpiled code only as adapter executable input. |
| Runtime ABI source hash and byte-size validation could be bypassed before adapter execution. | ABI bridge validates `source.text` hash and byte length before invoking the adapter. |
| Strategy Artifact schema accepted inconsistent built-in source/hash/bytes/eligibility metadata. | Artifact schema now enforces source hash, byte length, validation hash/bytes, and immutable eligibility source hash invariants. |
| Generated Advanced artifact lineage could point at the child hash instead of the parent artifact hash. | Manifest generation now resolves and verifies derived parent artifact hashes. |
| Go public Strategy/card outputs exposed private runtime limit internals. | Go public Strategy projections now use public runtime metadata and omit runtime limits. |
| Replay public projection allowed forbidden private marker strings in otherwise public JSON. | Replay public projection now sanitizes and asserts leak safety before returning public replay data. |
| Go privacy checks missed normalized private keys and forbidden marker strings. | Go privacy validation now normalizes keys and rejects expanded forbidden key/string markers. |
| v1.14 monitor coverage only checked fork routes for ownership metadata. | Boundary monitor now validates selected owner/fallback/evidence/disallowed scopes across all v1.14 ownership routes. |
| Replay realism checked bounds/piece sanity but not canonical start layout. | Replay preparation now validates canonical arena bounds, 16 canonical Soldiers, facings, and starting positions; live preflight replay was corrected to accept canonical `player:bottom` / `player:top` owner IDs. |
