# v1.21 Milestone Audit

**Milestone:** WASM/WASI Multi-Language Runtime Candidate and Rust Exhibition Alpha  
**Date:** 2026-05-25  
**Result:** Pass after remediation

## Scope

Audited v1.21 against the original goal: make WASM/WASI the next serious immutable multi-language Strategy runtime candidate, prove Rust end to end as non-counted exhibition alpha, keep Zig fail-loud, preserve JS/TS counted support, and avoid overclaiming production sandbox or ranked/counting support.

## Findings

No open milestone-blocking findings remain.

## Remediated During Audit

- Runtime-service Rust validation treated an empty `strategyId` as explicit metadata and failed Go-backed save. Fixed by omitting blank ids on both Go client and runtime-service validation.
- Account and MatchSet pages assumed every non-Python runtime was JS/TS. Fixed Rust/Zig-aware labels and evidence copy.
- The first live proof invocation accidentally ran both desktop and mobile projects. Reran desktop-only proof with the intended bounded command.

- Rust/WASM: non-counted exhibition alpha only.

- Python: remains non-counted exhibition beta.
