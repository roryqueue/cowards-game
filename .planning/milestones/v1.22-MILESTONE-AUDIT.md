## Evidence Checked

- Rust WASM/WASI path still compiles and executes through Wasmtime.
- Zig toolchain detection is PATH-aware and fails loud when unavailable.
- Zig compiles no-std source to immutable `wasm32-wasi` artifact metadata.
- Zig executes through runtime-service / Runtime Broker / Wasmtime using the existing Preview 1 stdin/stdout JSON ABI.
- WASM/WASI hardening probes pass 19/19.
- Artifact validation distinguishes Rust `wasm32-wasip1` from Zig `wasm32-wasi`.
- Workshop and account save paths require runtime-service validation for Rust and Zig.
- Public evidence language says candidate-readiness only, not production sandbox certification.
- ABI decision keeps Preview 1 stdin/stdout JSON as the only v1.22 execution path.
- Promotion decision keeps JS/TS counted, Python non-counted beta, Rust/Zig non-counted alpha only.
