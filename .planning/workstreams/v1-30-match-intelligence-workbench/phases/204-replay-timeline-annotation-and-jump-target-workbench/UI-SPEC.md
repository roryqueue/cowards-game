# Phase 204 UI Spec: Replay Timeline Intelligence

## Intent

Timeline intelligence should help players jump to public tactical moments without implying hidden Strategy intent.

## Requirements

- Annotation rows show category, sequence, Round, Activation, Cycle, confidence, and public evidence source.
- Filters use stable buttons and remain usable on mobile.
- Long annotation lists scroll internally so tactical panels stay reachable.
- Focus links use existing replay query behavior.

## Acceptance

- Desktop and mobile replay fixture proof can click a focus link and see the focus banner.
