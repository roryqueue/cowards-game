# Phase 205 UI Spec: Replay Tactical Panels

## Intent

Tactical panels should read as a serious inspection surface for autonomous Matches.

## Requirements

- Panels stay dense, bordered, and scannable.
- Board control copy states FALLEN Soldiers are excluded as occupants.
- Soldier status buttons expose status, last visible position, and public event involvement.
- Mobile layout keeps the board first, then timeline intelligence, then tactical panels.

## Acceptance

- No overlapping text or controls in desktop/mobile proof.
- Board canvas remains nonblank and in bounds.
