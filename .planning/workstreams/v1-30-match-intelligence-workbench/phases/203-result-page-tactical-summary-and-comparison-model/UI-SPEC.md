# Phase 203 UI Spec: Result Page Match Intelligence

## Intent

Result-page intelligence should feel like a compact tactical inspection panel, not marketing copy.

## Requirements

- Use the existing quiet operational visual language.
- Show evidence availability and confidence before tactical claims.
- Keep entrant comparison rows scannable and dense.
- Show no replay jump targets when public replay evidence is absent.
- Avoid private marker names in visible public output.

## Acceptance

- Desktop and mobile fixture pages render the panel without overlap.
- Every fixture has useful state-specific copy.
