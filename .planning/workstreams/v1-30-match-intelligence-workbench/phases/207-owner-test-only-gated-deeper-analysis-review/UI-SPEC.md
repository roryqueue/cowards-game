# Phase 207 UI Spec: Gated Diagnostics Boundary

## Intent

Default public intelligence must not look or behave like an owner-debug surface.

## Requirements

- No owner/debug controls in default public intelligence output.
- Owner-only Awareness Grid and inactivity explanations stay behind existing gates.
- Public intelligence copy references public evidence provenance, not private payload names.

## Acceptance

- Browser proof scans default result/replay output for private markers.
