# Phase 216 UI Spec

## Public Pages

- Public result pages must show evidence rows and privacy cues for every fixture state.
- Ready replay must show replay evidence and nonblank board rendering.
- Replay-unavailable pages must show reason, evidence rows, and privacy cue.

## Responsive Expectation

- Reuse existing responsive page shells and table/details styling.
- Avoid new decorative layout work; the milestone is trust polish, not a redesign.
