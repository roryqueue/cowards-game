# Phase 211 UI Spec

## Result Evidence Panel

- Add a compact `result state` row to the existing evidence panel.
- Keep dense operational layout; no landing-page or marketing treatment.
- Use existing table/details styling and avoid new DTO fields.

## Match Ledger

- When no replay link exists, show public labels such as `No public result`, `Invalid public result`, `System failure`, `Strategy failure`, `Running`, or `Pending`.
- Do not render raw enum names as player-facing proof.
