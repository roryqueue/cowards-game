# Phase 167 Plan

## Goal

Run and record abuse/no-fallback probes across Rust and Zig WASM/WASI non-counted exhibition beta lanes.

## Scope

In scope:
- Rust and Zig probes for timeout/fuel, memory cap, stdio/result cap, malformed JSON, invalid schema/action, panic/trap/abort, and runtime unavailability.
- Artifact probes for stale hash, missing artifact, mismatched target, mismatched runtime implementation, mismatched ABI metadata, and missing toolchain where applicable.
- Capability probes for forbidden filesystem/preopen, network, clock/random, environment, package, and host import surfaces where applicable.
- No-fallback proof that failed WASM/WASI lanes do not execute JS/TS, Python, mutable source, stale artifacts, or alternate ABIs.

Out of scope:
- Direct-export or Component Model/WIT Match execution.
- Any counted/ranked promotion for Rust, Zig, or WASM/WASI.

## Tasks

1. Map v1.24 taxonomy classes onto existing WASM/WASI hardening probes.
2. Extend Rust and Zig probe fixtures where coverage is missing.
3. Add mismatch and unavailable-lane probes for artifact metadata and ABI metadata.
4. Generate public-safe JSON and Markdown evidence.
5. Add checks proving Preview 1 stdin/stdout JSON remains the active execution ABI.
