# Phase 16 Plan 01: Manual Exhibition Entry

**Status:** Planned
**Requirements:** EXH-01, EXH-03, EXH-05, EXH-06

## Goal

Allow a signed-in User to create a manual exhibition MatchSet from 2-8 owned revisions.

## Tasks

1. Add account-owned revision selection API.
2. Add exhibition create endpoint with session auth.
3. Enforce owner-only revisions, 2-8 count, valid revisions, compatibility, preset rules, and distinct revision ids.
4. Lock snapshots and enqueue jobs on create.
5. Add tests for self-play and validation failures.

## Acceptance

- Same User can enter multiple distinct revision ids.
- Same revision id twice is rejected.
