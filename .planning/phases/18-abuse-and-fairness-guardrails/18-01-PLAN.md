# Phase 18 Plan 01: Rate Limits, Duplicate Guard, and Failure Policy

**Status:** Planned
**Requirements:** FAIR-01, FAIR-02, FAIR-03, FAIR-04

## Goal

Enforce basic abuse and fairness policy for exhibition creation and failure classification.

## Tasks

1. Add per-user create rate limit with small alpha burst.
2. Block active duplicate MatchSets for same creator/preset/normalized revision-id set.
3. Classify Strategy failures separately from system failures.
4. Apply deterministic affected-Match strategy failure penalties.
5. Add tests for rate limit, duplicate guard, and failure classification.

## Acceptance

- System failures never become player losses.
- Same active exhibition cannot be spammed while queued/running.
