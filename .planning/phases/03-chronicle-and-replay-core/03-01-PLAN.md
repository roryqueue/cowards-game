# Plan 03-01: Chronicle Contracts and Schemas

<objective>
Define the canonical Chronicle artifact, event context, boundary snapshot, integrity, validation, migration hook, and projection contracts in `@cowards/spec`.
</objective>

<requirements_addressed>
REPLAY-02, REPLAY-04, REPLAY-05, REPLAY-06, REPLAY-07
</requirements_addressed>

<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| T-03-01: Public and private Chronicle data are not structurally separable | high | Add explicit `ChroniclePublicEvent`, `ChroniclePrivateSections`, `ChronicleProjection`, and viewer contracts with schema tests. |
| T-03-02: Replay failures become untyped strings or raw throws | high | Add `ChronicleValidationErrorCode`, `ChronicleValidationError`, and version/migration contracts before implementation. |
| T-03-03: Contracts under-specify replay boundaries | medium | Add boundary snapshot kinds for MATCH_START, MATCH_END, ROUND_START, ROUND_END, ACTIVATION_START, ACTIVATION_END, CONTRACTION, and TERMINAL. |
</threat_model>

<tasks>
<task id="03-01-01" type="execute">
  <title>Add canonical Chronicle types</title>
  <read_first>
    - packages/spec/src/types.ts
    - packages/spec/src/schemas.ts
    - .planning/phases/03-chronicle-and-replay-core/03-CONTEXT.md
    - .planning/phases/03-chronicle-and-replay-core/03-RESEARCH.md
  </read_first>
  <action>
    Extend `packages/spec/src/types.ts` with:
    - `export type ChronicleSchemaVersion = "chronicle-v1"`
    - `export type ChronicleSnapshotKind = "MATCH_START" | "MATCH_END" | "ROUND_START" | "ROUND_END" | "ACTIVATION_START" | "ACTIVATION_END" | "CONTRACTION" | "TERMINAL"`
    - `export interface ChronicleEventContext { phaseNumber?: number; roundNumber?: 1 | 2 | 3 | 4; activationId?: string; activationIndex?: number; cycleIndex?: number; actingPlayerId?: PlayerId; soldierId?: SoldierId }`
    - `export type ChroniclePrivacy = "public" | "owner" | "private"`
    - `export interface ChronicleEvent { type: ChronicleEventType; sequence: number; context: ChronicleEventContext; privacy: ChroniclePrivacy; payload: JsonValue; privateRef?: string | undefined }`
    - `export interface ChronicleBoundarySnapshot { kind: ChronicleSnapshotKind; sequence: number; context: ChronicleEventContext; board: FullBoardSnapshot; outcome?: MatchOutcome | undefined }`
    - `export interface ChronicleReproducibilityEnvelope { matchId: MatchId; seed: string; arenaVariantId: ArenaVariantId; arenaVariantVersion: string; strategyRevisionIds: [StrategyRevisionId, StrategyRevisionId]; versions: CompatibilityVersions }`
    - `export interface ChronicleIntegrity { algorithm: "sha256"; normalizedContentHash: string }`
    - `export interface ChroniclePrivateSections { byPlayerId: Record<PlayerId, JsonValue>; debug?: JsonValue | undefined }`
    - `export interface Chronicle { schemaVersion: ChronicleSchemaVersion; reproducibility: ChronicleReproducibilityEnvelope; events: ChronicleEvent[]; snapshots: ChronicleBoundarySnapshot[]; private?: ChroniclePrivateSections | undefined; integrity?: ChronicleIntegrity | undefined; storageMetadata?: JsonValue | undefined }`
    - `export type ChronicleValidationErrorCode = "SCHEMA_INVALID" | "VERSION_INCOMPATIBLE" | "EVENT_ORDER_INVALID" | "REQUIRED_EVENT_MISSING" | "SNAPSHOT_MISSING" | "SNAPSHOT_MISMATCH" | "HASH_MISMATCH" | "PRIVATE_ACCESS_DENIED" | "UNSUPPORTED_MIGRATION"`
    - `export interface ChronicleValidationError { code: ChronicleValidationErrorCode; sequence?: number | undefined; message: string; expected?: JsonValue | undefined; actual?: JsonValue | undefined }`
    - `export type ChronicleValidationResult = { ok: true } | { ok: false; errors: ChronicleValidationError[] }`
    - projection/viewer types for `"public"` and `"owner"` access.
  </action>
  <verify>
    pnpm --filter @cowards/spec typecheck
    rg "interface Chronicle " packages/spec/src/types.ts
    rg "ChronicleBoundarySnapshot" packages/spec/src/types.ts
    rg "ChronicleValidationErrorCode" packages/spec/src/types.ts
  </verify>
  <acceptance_criteria>
    - `packages/spec/src/types.ts` contains `export interface Chronicle`
    - `packages/spec/src/types.ts` contains `export interface ChronicleBoundarySnapshot`
    - `packages/spec/src/types.ts` contains `ChronicleValidationErrorCode`
    - `packages/spec/src/types.ts` contains `ChronicleReproducibilityEnvelope`
    - `pnpm --filter @cowards/spec typecheck` exits 0
  </acceptance_criteria>
</task>
