# Phase 15 Plan 02: Pairwise Matrix and Scoring Policy

**Status:** Planned
**Requirements:** COMP-06, COMP-07

## Goal

Implement pairwise mirrored Match composition and explicit competition scoring.

## Tasks

1. Generate all distinct entrant pairs for selected revisions.
2. Apply preset arena/seed/mirror policy to every pair.
3. Add points: win 3, draw 1, loss 0, with strategy failure penalty support.
4. Add deterministic tie-breaker details: points/wins, head-to-head, surviving Soldiers, survival turns, revision hash fallback.
5. Add tests for multi-entrant scoring and deterministic ordering.

## Acceptance

- No tie-breaker depends on clock time, row order, or worker scheduling.
- Public scoring DTO can explain why standings are ordered.
