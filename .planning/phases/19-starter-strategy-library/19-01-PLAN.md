# Phase 19 Plan: Starter Strategy Library

## Scope

1. Add 10 credible Starter Strategies with doctrine metadata, tags, source hashes, validation status, and readable source.
2. Add persistence APIs to list starters and fork one into an account-owned Strategy Revision.
3. Add Workshop UI for browsing, applying, and forking starters.
4. Preserve starter lineage on matching saved source and account revision summaries.
5. Add validation and Workshop tests.

## Acceptance

- Starters: Centerline Bully, Corner Lurker, Backstab Hunter, Wall Press, Ring Runner, Mirror Breaker, Center Turtle, Aggro Chaser, Escape Artist, Trap Setter.
- At least three starters use memory.
- Starters are not automatically submitted to competition.
