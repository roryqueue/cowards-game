# Phase 19 UI Spec

## Contract

- Workshop shows a Starter Library section with starter name, description, tags, validation state, and source hash.
- Applying a starter loads source into the editor without creating account data.
- Forking a starter creates a normal account-owned revision.
- Editing applied starter source clears starter lineage until the player explicitly forks or saves matching source.

## Privacy

Starter source is public template source. Account-owned source remains owner-only unless explicitly shown in the Workshop.
