<threat_model>
## Trust Boundaries

| Boundary | Description |
| --- | --- |
| parent worker -> subprocess stdin | Host sends untrusted Strategy source and inputs over JSON IPC. |
| subprocess stdout -> parent worker | Child output crosses back into trusted orchestration. |
| child process -> host OS | Child has process-level access that must be minimized. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
| --- | --- | --- | --- | --- |
| T-10-04 | T/E | subprocess spawn | mitigate | Use no shell, minimal env, bounded stdio, and timeout kill. |
| T-10-05 | D | subprocess stdout/stderr | mitigate | Enforce byte caps before JSON parsing. |
| T-10-06 | S/T | IPC response | mitigate | Schema-validate JSON response and throw system failures for malformed IPC. |
| T-10-07 | I | environment inheritance | mitigate | Do not pass `process.env`; use explicit minimal env only. |
</threat_model>
